## Getting Started

### Add to your project

Make sure to update `VERSION` to the latest version of the api.

> Maven
```xml

  <groupId>net.icxd</groupId>
  <artifactId>mongo-utils</artifactId>
  <version>VERSION</version>
</dependency>
```
